## 💜 주요기능

- 기능 1 : 커머스 플랫폼 실행
  - 주요 내용 : 각 판매 카테고리 출력, 카테고리 선택, 종료 
- 기능 2 : 선택된 카테고리 상품내역 출력
  - 주요 내용 : 카테고리별 상품내역 출력, 상품 선택, 뒤로가기
- 기능 3 : 선택된 상품 출력
  - 주요 내용 : 선택된 상품 출력 후 상품을 선택하여 장바구니에 담을 수 있습니다.
- 기능 4 : [주문관리]( 장바구니에 상품을 담았다면 활성화 ) 장바구니 확인 기능
  - 주요 내용 : 1. 장바구니에 들어있는 상품을 확인할수 있습니다. 2. 주문 확정을 할 수 있습니다.)
- 기능 5 : [주문관리]( 장바구니에 상품을 담았다면 활성화 ) 주문 취소 기능
  - 주요 내용 : 장바구니에 담았던 상품들이 전체 취소가 됩니다.
- 기능 6 : 관리자 모드(비밀번호 : admin)
  - 주요 내용 : 1. 관리자 로그인에 성공하면 관리자 모드를 사용할수 있습니다. 비밀번호 3회 이상 틀릴 시 종료
              2. 상품 추가, 수정, 삭제, 전체 상품 현황 조회가 가능합니다. 상품 삭제 시 장바구니에 담긴 상품도 같이 삭제됩니다.
